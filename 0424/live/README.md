# EVENT

### 개요
- 이벤트란 HTML 요소에서 발생하는 모든 상황을 의미
- 웹페이지의 버튼을 클릭한다면 결과를 받거나 조작을 할 수 있음
- 키보키 이 입력, 브라우저닫기, 데이터 제출, 텍스트 복사 등

### Event object
- DOM 요소는 이벤트를 받고 받은 EVENT를 처리 할 수 있음
